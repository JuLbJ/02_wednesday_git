Git toutorial for real beginners

Git is full of features but we will cover just the basics here. Follow instractions - 'cause that's the kernel of the git.

  • for the git installation ask your friend for help - you are the beginner - it will save your time :)
  • for the real work use the advices below
  • if any questions search: "Troubleshooting"

Create an environment of your work and check if the git is for you.

Create a new repository = Follow instructions

note:"test" is the name of your repository

  • echo "# test01" >> README.md
    • create the folder on your terminal
    • create the file in your new folder and name it: README.md
  • git init
  • git add README.md
  • git commit -m "first commit"
  • git remote add origin git@github.com:JuLbJ/test01.git
  • git push -u origin master
